## 형식 선택

### EPUB

전자책 표준 형식입니다.

[사진: EPUB 선택]

| 특징 | 설명 |
|------|------|
| 목차 지원 | 자동 목차 생성 |
| 메타데이터 | 제목, 작가 정보 포함 |
| 호환성 | 대부분의 전자책 리더 지원 |

**추천**: 전자책 리더 (Kindle, 리디북스 등)에서 읽을 때

### TXT

단순 텍스트 형식입니다.

[사진: TXT 선택]

| 특징 | 설명 |
|------|------|
| 단순함 | 텍스트만 포함 |
| 범용성 | 모든 텍스트 편집기에서 열기 |
| 가벼움 | 파일 크기 최소 |

**추천**: 편집이 필요하거나 다른 용도로 활용할 때

### HTML

웹 페이지 형식입니다.

[사진: HTML 선택]

| 특징 | 설명 |
|------|------|
| 브라우저 | 웹 브라우저에서 열기 |
| 스타일링 | 기본 스타일 포함 |
| 링크 | 회차 간 링크 |

**추천**: 브라우저에서 열람하거나 웹에 게시할 때

## 텍스트 유형 선택

내보낼 텍스트의 종류를 선택합니다.

[사진: 텍스트 유형 선택]

### 원문

번역 전 원본 텍스트만 내보냅니다.

### 번역

번역된 텍스트만 내보냅니다.

### 원문+번역

원문과 번역을 함께 내보냅니다.

```
【원문】
これは日本語のテストです。

【번역】
이것은 일본어 테스트입니다.
```

> **참고**: EPUB 형식에서는 "원문+번역" 옵션이 제한될 수 있습니다.

## 회차 범위 선택

내보낼 회차 범위를 지정합니다.

[사진: 회차 범위 선택]

### 시작/종료 회차

- 시작 회차: 첫 번째로 포함할 회차
- 종료 회차: 마지막으로 포함할 회차

### 전체 선택

모든 회차를 내보냅니다.

### 번역 상태 미리보기

선택한 범위의 번역 상태를 확인할 수 있습니다.

```
선택 범위: 1-100회차
번역 완료: 95회차
미번역: 5회차 (96, 97, 98, 99, 100)
```

## 문제 해결

### 내보내기가 실패함

**확인**:
1. 저장 공간 충분한지
2. 권한 허용되었는지
3. 선택한 회차에 데이터가 있는지

### 파일이 열리지 않음

**해결**:
1. 올바른 앱으로 열기
2. 파일이 손상되지 않았는지 확인
3. 다시 내보내기 시도

### EPUB이 특정 리더에서 안 열림

**해결**:
1. 다른 EPUB 리더 시도
2. TXT로 내보내기 후 변환

### 글자가 깨져 보임

**원인**: 인코딩 문제

**해결**:
1. UTF-8 지원 앱 사용
2. 텍스트 편집기에서 인코딩 변경

### 내보내기가 너무 느림

**해결**:
1. 더 적은 회차 선택
2. 저장 공간 확보
3. 앱 재시작 후 시도
