## TTS 비교

| 항목 | Android TTS | Gemini TTS |
|------|-------------|------------|
| **품질** | 기본 | 고품질 |
| **비용** | 무료 | API 비용 발생 |
| **오프라인** | 지원 | 미지원 |
| **속도 조절** | 지원 | 지원 |
| **캐싱** | 미지원 | 지원 |
| **스타일** | 고정 | 커스터마이징 가능 |
