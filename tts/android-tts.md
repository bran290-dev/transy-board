# Android TTS

Android 기기에 내장된 TTS 엔진을 사용하는 방법입니다.

---

## Android TTS란?

Android 기기에 기본으로 포함된 음성 합성 엔진입니다.

### 특징

| 특징 | 설명 |
|------|------|
| **무료** | 추가 비용 없음 |
| **오프라인** | 인터넷 연결 불필요 |
| **빠른 시작** | 즉시 재생 가능 |
| **배터리 효율** | 적은 배터리 소모 |

---

## 사전 설정

### 시스템 TTS 설정

Android TTS를 사용하려면 시스템 설정이 필요합니다.

1. 기기 설정 열기
2. 접근성 > 텍스트 읽어주기 (또는 TTS)
3. 원하는 TTS 엔진 선택

[사진: 시스템 TTS 설정 화면]

### 언어 데이터 다운로드

한국어 음성 데이터가 필요합니다.

1. TTS 설정에서 한국어 선택
2. 음성 데이터 다운로드

[사진: 언어 데이터 다운로드]

> **팁**: Google TTS 앱을 사용하면 고품질 음성을 사용할 수 있습니다.

## 자동 다음 회차

현재 회차가 끝나면 자동으로 다음 회차로 이동합니다.

### 설정

1. 뷰어 설정 > TTS
2. "자동 다음 회차" 토글 ON

[사진: 자동 다음 회차 설정]

### 동작

1. 현재 회차 마지막 문장까지 읽기
2. 짧은 대기 (1-2초)
3. 다음 회차로 자동 전환
4. 읽기 계속

## Android TTS 설정 상세

### 앱 내 설정

설정 > TTS에서 조정:

| 설정 | 설명 | 기본값 |
|------|------|--------|
| TTS 제공자 | Android/Gemini 선택 | Android |
| 속도 | 읽기 속도 | 1.0x |
| 피치 | 음성 높낮이 | 1.0 |
| 자동 재생 | 회차 열 때 자동 시작 | OFF |

### 시스템 설정 권장

| 설정 | 권장값 |
|------|--------|
| TTS 엔진 | Google TTS |
| 언어 | 한국어 |
| 음성 품질 | 고품질 |

## 장점과 한계

### 장점

- 무료 사용
- 오프라인 사용 가능
- 빠른 시작
- 배터리 효율적

### 한계

- 음성 품질이 Gemini보다 낮음
- 억양이 부자연스러울 수 있음
- 캐싱 미지원
- 스타일 커스터마이징 불가

## 문제 해결

### "TTS를 사용할 수 없습니다"

**해결**:
1. 시스템 TTS 설정 확인
2. TTS 엔진 설치 확인
3. 언어 데이터 다운로드 확인

### 음성이 나오지 않음

**해결**:
1. 미디어 볼륨 확인
2. 블루투스 연결 상태 확인
3. 앱 재시작

### 읽기가 중간에 멈춤

**해결**:
1. 배터리 절약 모드 해제
2. 앱 배터리 최적화 제외

### 한국어로 읽히지 않음

**해결**:
1. 번역된 텍스트인지 확인
2. 시스템 TTS 언어 설정 확인
3. 한국어 음성 데이터 다운로드
