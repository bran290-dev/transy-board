뷰어 화면에서 설정 버튼(⚙️)을 클릭합니다.

---

## 폰트 설정

### 폰트 크기

텍스트 크기를 조절합니다.

[사진: 폰트 크기 슬라이더]

| 크기 | 용도 |
|------|------|
| 12-14sp | 작은 화면, 많은 텍스트 |
| 16-18sp | 일반 읽기 (권장) |
| 20-24sp | 큰 글씨 선호, 눈 피로 감소 |

### 폰트 패밀리

사용할 폰트를 선택합니다.

[사진: 폰트 선택]

| 옵션 | 설명 |
|------|------|
| 기본 | 시스템 기본 폰트 |
| 명조 | 명조체 스타일 (세리프) |
| 고딕 | 고딕체 스타일 (산세리프) |
| 손글씨 | 손글씨 스타일 |

### 폰트 굵기

텍스트의 굵기를 조절합니다.

[사진: 폰트 굵기 옵션]

| 옵션 | 설명 |
|------|------|
| 가벼움 | 얇은 굵기 |
| 보통 | 일반 (기본) |
| 중간 | 중간 굵기 |
| 굵게 | 굵은 굵기 |

---

## 간격 설정

### 줄 간격

줄 사이의 간격을 조절합니다.

[사진: 줄 간격 슬라이더]

| 값 | 설명 |
|------|------|
| 1.0 | 촘촘함 |
| 1.5 | 일반 (기본) |
| 2.0 | 넓음 |

> **팁**: 줄 간격이 넓으면 가독성이 좋지만 스크롤이 많아집니다.

### 문단 간격

문단 사이의 간격을 조절합니다.

[사진: 문단 간격 옵션]

| 옵션 | 설명 |
|------|------|
| 좁게 | 최소 간격 |
| 보통 | 일반 (기본) |
| 넓게 | 넓은 간격 |

## 화면 설정

### 화면 유지

읽는 동안 화면이 자동으로 꺼지지 않게 합니다.

[사진: 화면 유지 토글]

| 상태 | 동작 |
|------|------|
| OFF | 시스템 설정에 따라 화면 꺼짐 |
| ON | 뷰어에서 화면이 꺼지지 않음 |

> **주의**: 배터리 소모가 증가합니다.

## 권장 설정

### 일반 읽기

| 설정 | 값 |
|------|------|
| 폰트 크기 | 16sp |
| 폰트 패밀리 | 기본 |
| 줄 간격 | 1.5 |
| 문단 간격 | 보통 |
| 화면 유지 | ON |

### 장시간 읽기

| 설정 | 값 |
|------|------|
| 폰트 크기 | 18sp |
| 폰트 굵기 | 중간 |
| 줄 간격 | 1.8 |
| 문단 간격 | 넓게 |
| 화면 유지 | ON |

### 어두운 환경

| 설정 | 값 |
|------|------|
| 폰트 크기 | 18sp |
| 폰트 굵기 | 중간 |
| 시스템 테마 | 다크 모드 |

## 문제 해결

### 폰트가 적용 안 됨

**해결**:
1. 앱 재시작
2. 다른 폰트 선택 후 다시 변경

### 설정이 저장 안 됨

**해결**:
1. 앱 권한 확인
2. 저장 공간 확인
3. 앱 재설치

### 화면 유지가 작동 안 함

**해결**:
1. 시스템 배터리 절약 모드 확인
2. 앱 배터리 최적화 제외 설정
