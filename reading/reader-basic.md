## 원문/번역 전환

번역된 상태에서 원문과 번역문을 전환할 수 있습니다.

[사진: 원문/번역 전환 버튼]

### 표시 모드

| 모드 | 설명 |
|------|------|
| **번역문** | 번역된 텍스트 표시 (기본) |
| **원문** | 원본 텍스트 표시 |

전환 버튼(🌐)을 클릭하면 두 모드 사이를 토글합니다.

> **팁**: 번역 품질을 확인하거나 원문 표현을 참고할 때 유용합니다.

---

## 인증이 필요한 사이트

일부 소설 사이트는 로그인이나 캡차 인증이 필요합니다.

### 자동 인증 안내

원문 다운로드 시 인증이 필요하면 안내 다이얼로그가 표시됩니다.

[사진: 인증 필요 안내]

### 인증 방법

1. "수동 확인" 버튼 클릭
2. WebView에서 로그인 또는 캡차 해결
3. 인증 완료 후 자동으로 콘텐츠 로드

[사진: 인증 WebView]

### 지원하는 인증 유형

| 유형 | 설명 |
|------|------|
| 로그인 | 사이트 계정 로그인 |
| Cloudflare | 보안 검사 통과 |
| 캡차 | 이미지/reCAPTCHA 등 |

## 제스처 조작

### 스크롤

- **위/아래 스와이프**: 텍스트 스크롤

### 탭

- **캐릭터 이름 탭**: 캐릭터 정보 표시

[사진: 길게 누르기 메뉴]

텍스트를 길게 누르면:
- **복사**: 선택한 텍스트 복사
- **드래그 선택**: 범위 선택 모드
- **TTS 시작점**: 해당 위치부터 TTS 시작

## 회차 이동

### 이전/다음 버튼

하단 바의 ◀/▶ 버튼으로 회차를 이동합니다.

### 회차 목록

[사진: 회차 목록 패널]

1. 상단의 목록 버튼(📋) 클릭
2. 전체 회차 목록 표시
3. 원하는 회차 선택

### 회차 상태 표시

| 상태 | 표시 |
|------|------|
| 미다운로드 | 흐리게 표시 |
| 다운로드됨 | 일반 표시 |
| 번역됨 | ✓ 표시 |
| 현재 회차 | 강조 표시 |

## 번역 삭제

잘못된 번역을 삭제하고 다시 번역할 수 있습니다.

1. 메뉴(⋮) 열기
2. "번역 삭제" 선택
3. 확인 후 삭제
4. 번역 버튼으로 다시 번역

## 유용한 팁

### 끊김 없이 읽기

- "미리 받기" 기능을 활성화하면 다음 회차를 미리 준비
- 회차 전환 시 로딩 대기 시간 감소

### 장시간 읽기

- "화면 유지" 옵션을 켜면 화면이 꺼지지 않음
- 배터리 소모에 주의

### 눈 피로 감소

- 어두운 환경에서는 시스템 다크 모드 사용
- 적절한 폰트 크기 설정 (권장: 16-18sp)

---

## 문제 해결

### 텍스트가 너무 작음/큼

- 설정에서 폰트 크기 조절
- 시스템 폰트 크기 설정도 확인

### 원문 다운로드 실패

**해결**:
1. 인터넷 연결 확인
2. 사이트 접근 가능한지 확인
3. 인증이 필요한 경우 "수동 확인" 진행
4. "원문 재다운로드" 시도

### 캐릭터 하이라이트가 안 됨

- 캐릭터가 컨텍스트에 등록되어 있는지 확인
- 이름 철자가 일치하는지 확인
- 번역문 표시 모드인지 확인 (원문 모드에서는 미작동)

### 인증이 계속 필요함

- 파싱 룰에 인증 설정이 올바른지 확인
- 브라우저 쿠키가 유지되는지 확인
- 앱 재시작 후 다시 인증
