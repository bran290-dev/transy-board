# 일반적인 문제 해결

Transy 사용 중 발생할 수 있는 문제와 해결 방법입니다.

---

## 번역 관련

### 번역이 시작되지 않음

**증상**: 번역 버튼을 눌러도 반응이 없음

**해결 방법**:
1. API 키가 등록되어 있는지 확인
   - 설정 > API 키 관리
2. 인터넷 연결 상태 확인
3. 원문이 다운로드되었는지 확인
4. 앱 재시작

---

### "API 키가 유효하지 않습니다"

**증상**: 번역 시 API 키 오류 메시지

**해결 방법**:
1. API 키 앞뒤 공백 확인
   - 설정 > API 키 관리 > 키 재입력
2. Google AI Studio에서 키 상태 확인
   - 만료, 삭제 여부
3. 새 API 키 발급 후 재등록

---

### "할당량을 초과했습니다"

**증상**: API 사용량 한도 도달

**해결 방법**:
1. 무료 티어 한도 확인
   - 일일 한도 리셋 대기 (24시간)
2. 다른 모델로 API 설정 체인 구성
   - 설정 > API 설정
3. 유료 플랜으로 업그레이드

---

### 번역이 중간에 멈춤

**증상**: 번역 진행 중 멈추거나 실패

**해결 방법**:
1. API 타임아웃 증가
   - 소설별 설정 > LLM 타임아웃
2. 네트워크 상태 확인
3. 번역 삭제 후 재시도

---

### 번역 품질이 낮음

**증상**: 어색한 번역, 오역

**해결 방법**:
1. 컨텍스트 시스템 활용
   - 캐릭터/용어 등록
2. 번역 프롬프트 조정
   - 소설별 설정 > 번역 프롬프트
3. 더 높은 품질의 AI 모델 사용
   - Gemini 2.5 Pro

---

### 캐릭터 이름이 일관되지 않음

**증상**: 같은 캐릭터가 다른 이름으로 번역됨

**해결 방법**:
1. 캐릭터 컨텍스트에 등록
   - 컨텍스트 관리 > 캐릭터
2. 별명 모두 등록
3. 자동 보정 활성화
   - 소설별 설정 > 캐릭터 자동 보정
4. 일괄 치환으로 기존 번역 수정

---

## 소설 추가 관련

### URL 파싱 실패

**증상**: URL을 입력해도 소설 정보가 추출되지 않음

**해결 방법**:
1. URL이 올바른지 확인
   - 회차 목록 페이지 URL 사용
   - 개별 회차 URL 아님
2. 사이트 접근 가능한지 확인
3. 파싱 룰 확인
   - 해당 사이트 지원 여부
4. 파싱 룰 생성
   - 설정 > 파싱 룰 생성기

---

### 로그인 필요 사이트 접근 실패

**증상**: 로그인이 필요한 사이트에서 콘텐츠를 가져올 수 없음

**해결 방법**:
1. 파싱 룰 생성기에서 WebView 인증 사용
   - 파싱 룰 생성 과정에서 WebView 창이 열림
   - 해당 사이트에 직접 로그인
2. 인증 후 파싱 룰 저장
3. 저장된 쿠키로 이후 접근 가능

---

### CAPTCHA 인증 필요

**증상**: CAPTCHA 확인이 필요하다는 오류

**해결 방법**:
1. 파싱 룰 생성기에서 해당 사이트 열기
2. WebView에서 CAPTCHA 직접 풀기
3. 인증 완료 후 파싱 진행

---

### 세션 만료

**증상**: 이전에 접근 가능했던 사이트에서 인증 오류

**해결 방법**:
1. 파싱 룰 생성기에서 해당 사이트 다시 열기
2. 재로그인하여 세션 갱신
3. 파싱 룰 업데이트

---

### 회차 목록이 불완전함

**증상**: 일부 회차만 추출됨

**해결 방법**:
1. 파싱 룰 업데이트 필요
2. 사이트 구조 변경 가능성
3. 새로고침으로 나머지 회차 추가 시도

---

### 파일 불러오기 실패

**증상**: TXT/EPUB 파일을 선택해도 불러와지지 않음

**해결 방법**:
1. 파일 형식 확인 (TXT, EPUB)
2. 파일 손상 여부 확인
3. 저장소 권한 확인
4. 파일 인코딩 확인 (UTF-8 권장)

---

## TTS 관련

### Android TTS가 작동하지 않음

**증상**: TTS 버튼을 눌러도 음성이 나오지 않음

**해결 방법**:
1. 시스템 TTS 설정 확인
   - 기기 설정 > 접근성 > TTS
2. TTS 엔진 설치 확인
   - Google TTS 앱 설치
3. 한국어 음성 데이터 다운로드
4. 미디어 볼륨 확인

---

### Gemini TTS 실패

**증상**: "오디오 생성에 실패했습니다"

**해결 방법**:
1. Gemini API 키 확인
2. 인터넷 연결 확인
3. TTS 타임아웃 증가
   - 소설별 설정 > TTS 타임아웃
4. 청크 크기 조정

---

### TTS가 끊기거나 멈춤

**증상**: 재생 중 끊김 또는 멈춤

**해결 방법**:
1. 네트워크 상태 확인 (Gemini TTS)
2. 배터리 절약 모드 해제
3. 앱 백그라운드 실행 허용

---

## 앱 성능 관련

### 앱이 느림

**증상**: 전반적인 앱 성능 저하

**해결 방법**:
1. 앱 캐시 정리
2. 불필요한 소설 삭제
3. 앱 재시작
4. 기기 재시작

---

### 저장 공간 부족

**증상**: 저장 공간 경고

**해결 방법**:
1. TTS 캐시 정리
2. 읽은 소설의 번역 캐시 정리
3. 사용하지 않는 소설 삭제

---

### 앱 크래시

**증상**: 앱이 갑자기 종료됨

**해결 방법**:
1. 앱 업데이트 확인
2. 앱 캐시 삭제
3. 앱 재설치
4. 문제 지속 시 피드백 보내기

---

## 내보내기 관련

### 내보내기 실패

**증상**: 내보내기가 완료되지 않음

**해결 방법**:
1. 저장 공간 확인
2. 권한 허용 확인
3. 더 적은 회차로 시도
4. 다른 형식으로 시도

---

### 내보낸 파일이 열리지 않음

**증상**: 파일은 생성되었으나 열 수 없음

**해결 방법**:
1. 올바른 앱으로 열기
   - EPUB: 전자책 리더
   - TXT: 텍스트 편집기
   - HTML: 웹 브라우저
2. 다시 내보내기 시도

---

## 기타 문제

### 설정이 저장되지 않음

**증상**: 설정 변경이 유지되지 않음

**해결 방법**:
1. 저장 공간 확인
2. 앱 권한 확인
3. 앱 재설치

---

### 알림이 오지 않음

**증상**: 백그라운드 작업 알림 없음

**해결 방법**:
1. 알림 권한 확인
2. 배터리 최적화에서 제외
3. 알림 채널 설정 확인

---

## 피드백 보내기

문제가 해결되지 않으면:

1. **GitHub Issues**에서 문제 보고
2. 다음 정보 포함:
   - 앱 버전
   - 기기 모델
   - Android 버전
   - 문제 재현 단계
   - 오류 메시지 (있는 경우)

---

## 앱 초기화

모든 문제 해결 방법이 실패한 경우:

> **주의**: 모든 데이터가 삭제됩니다.

1. 앱 데이터 백업 (내보내기)
2. 설정 > 앱 > Transy > 저장공간 > 데이터 삭제
3. 앱 재설정
