# 소설 업데이트

연재 중인 소설의 새 회차를 가져오는 방법입니다.

---

## 새로고침 기능

URL로 추가한 소설은 원본 사이트에서 새 회차를 확인하고 자동으로 추가할 수 있습니다.

[사진: 업데이트 확인 버튼]

> **참고**: 파일이나 텍스트로 추가한 소설은 새로고침 기능을 사용할 수 없습니다.

### 방법 1: 라이브러리에서

1. 라이브러리에서 소설 카드를 길게 누르기
2. "회차 업데이트 확인" 선택

[사진: 라이브러리 컨텍스트 메뉴]

### 방법 2: 상세 화면에서

1. 소설 상세 화면으로 이동
2. 상단의 새로고침 아이콘(🔄) 클릭

[사진: 상세 화면 업데이트 확인 버튼]

---

## 새로고침 과정

[사진: 새로고침 진행 중]

### 단계

1. **URL 접속** - 원본 사이트에 연결
2. **파싱** - 회차 목록 추출
3. **비교** - 기존 회차와 비교
4. **추가** - 새 회차 저장

### 소요 시간

- 일반적으로 수 초 내 완료
- 사이트 응답 속도에 따라 다름

## 자동 새로고침

### 현재 지원 상태

현재 버전에서는 자동 새로고침 기능이 없습니다. 수동으로 새로고침해 주세요.

### 추천 사용 방법

- 읽기 전에 새로고침 실행
- 연재일에 맞춰 확인

## 문제 해결

### "새로고침에 실패했습니다"

**원인**:
- 인터넷 연결 문제
- 원본 사이트 접속 불가
- 파싱 룰 문제

**해결**:
1. 인터넷 연결 확인
2. 브라우저에서 원본 사이트 접근 가능한지 확인
3. 잠시 후 다시 시도

### 새 회차가 있는데 감지 안 됨

**원인**:
- 사이트 구조 변경
- 파싱 룰 업데이트 필요

**해결**:
1. 앱 업데이트 확인
2. 파싱 룰 재생성 필요할 수 있음

### 잘못된 회차가 추가됨

**원인**:
- 사이트에서 공지사항 등이 회차로 인식됨
- 파싱 룰 정확도 문제

**해결**:
1. 잘못된 회차는 수동으로 삭제
2. 파싱 룰 수정 필요

## URL 소설 관리 팁

### 연재 상태 확인

- 완결된 소설은 더 이상 새로고침 필요 없음
- 연재 중인 소설은 정기적으로 확인

### 여러 소설 관리

- 연재 중인 소설을 우선적으로 관리
- 완결된 소설은 별도 폴더로 정리 (향후 기능)

### 데이터 사용량

- 새로고침은 회차 목록만 가져옴 (적은 데이터)
- 실제 본문은 읽을 때 다운로드
