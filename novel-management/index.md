# 소설 관리

Transy에서 소설을 추가하고 관리하는 방법을 안내합니다.

---

## 목차

1. [URL로 소설 추가](add-novel-url.md) - 웹소설 사이트에서 소설 가져오기
2. [파일로 소설 추가](add-novel-file.md) - TXT/EPUB 파일 불러오기
3. [텍스트 직접 입력](add-novel-text.md) - 텍스트를 직접 입력하여 추가
4. [라이브러리 관리](library.md) - 소설 목록 관리 및 검색
5. [소설 업데이트](novel-refresh.md) - 연재 중인 소설 새 회차 가져오기

## 소설 추가 방식 비교

| 방식 | 장점 | 단점 | 추천 상황 |
|------|------|------|----------|
| **URL** | 자동 업데이트, 메타데이터 자동 추출 | 파싱 룰 필요 | 연재 중인 웹소설 |
| **파일** | 오프라인 사용 가능 | 업데이트 불가 | 완결작, 보유 파일 |
| **텍스트** | 빠른 테스트 가능 | 대량 입력 불편 | 테스트, 짧은 글 |
