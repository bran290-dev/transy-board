# 라이브러리 관리

라이브러리는 Transy의 메인 화면으로, 등록된 모든 소설을 관리합니다.

---

## 라이브러리 화면

[사진: 라이브러리 메인 화면]

### 화면 구성

| 영역 | 설명 |
|------|------|
| 상단 바 | 앱 타이틀, 검색, 설정 버튼 |
| 소설 목록 | 등록된 소설 카드 목록 |
| 하단 버튼 | 소설 추가 (+) 버튼 |

---

## 소설 카드

각 소설은 카드 형태로 표시됩니다.

[사진: 소설 카드 상세]

### 표시 정보

| 항목 | 설명 |
|------|------|
| 제목 | 소설 제목 |
| 작가 | 작가명 (있는 경우) |
| 진행도 | 읽은 회차 / 전체 회차 |
| 최근 업데이트 | 최근 업데이트 시간 |
| 출처 아이콘 | URL/파일/텍스트 구분 |

### 진행도 표시

```
12/156 회차 (7%)
━━━━━░░░░░░░░░░░░░░░
```

## 소설 검색

상단에 항상 표시되는 검색바에서 소설을 검색할 수 있습니다.

[사진: 검색 바]

### 검색 대상

- 소설 제목
- 작가명

### 사용 방법

1. 검색바에 검색어 입력
3. 실시간으로 필터링된 결과 표시
4. X 버튼으로 검색 종료

---

## 소설 관리 메뉴

소설 카드에 관리 버튼들이 직접 표시됩니다.

[사진: 소설 카드 버튼들]

### 메뉴 항목

| 메뉴 | 설명 |
|------|------|
| **제목 편집** | 소설 제목 수정 |
| **새로고침** | 새 회차 확인 (URL 소설만) |
| **삭제** | 소설 삭제 |

---

## 제목 편집

소설 제목을 원하는 대로 수정할 수 있습니다.

[사진: 제목 편집 다이얼로그]

1. 소설 카드의 편집(연필) 아이콘 클릭
3. 새 제목 입력
4. "확인" 클릭

> **참고**: 원본 출처의 제목은 변경되지 않습니다.

---

## 소설 새로고침

URL로 추가한 소설은 새 회차를 확인할 수 있습니다.

### 개별 새로고침

1. 소설 카드의 새로고침 아이콘 클릭
3. 새 회차 확인 및 추가

[사진: 새로고침 진행]

### 새로고침 결과

- 새 회차가 있으면 자동 추가
- 새 회차가 없으면 "업데이트 없음" 표시

> **참고**: 파일/텍스트로 추가한 소설은 새로고침 불가

---

## 소설 삭제

소설을 라이브러리에서 삭제합니다.

[사진: 삭제 확인 다이얼로그]

### 삭제 시 제거되는 데이터

- 소설 정보 (제목, 작가, 설명)
- 모든 회차 데이터
- 번역 캐시
- 컨텍스트 정보 (캐릭터, 용어 등)
- TTS 오디오 캐시

> **주의**: 삭제된 데이터는 복구할 수 없습니다.

### 삭제 방법

1. 소설 카드의 삭제 아이콘 클릭
3. 확인 다이얼로그에서 "삭제" 클릭

## 빠른 접근

### 소설 상세 화면으로

소설 카드를 탭하면 상세 화면으로 이동합니다.

### 이어 읽기

소설 카드의 "이어 읽기" 버튼을 탭하면 마지막 읽던 회차로 바로 이동합니다.

[사진: 이어 읽기 버튼]

#### 이어 읽기 동작

- **마지막 읽던 회차** 자동 기억
- **스크롤 위치**도 함께 저장되어 읽던 부분부터 시작
- 여러 소설을 번갈아 읽어도 각각의 위치 유지

#### 유용한 상황

| 상황 | 동작 |
|------|------|
| 처음 읽는 소설 | 1화부터 시작 |
| 읽던 소설 재개 | 마지막 위치로 이동 |
| 완독한 소설 | 마지막 회차 표시 |

> **팁**: 여러 소설을 동시에 읽고 있을 때, 라이브러리에서 "이어 읽기"를 사용하면 각 소설의 마지막 위치로 빠르게 돌아갈 수 있습니다.

---

## 문제 해결

### 소설이 표시되지 않음
- 검색 필터가 활성화되어 있는지 확인
- 앱을 재시작해 보세요

### 카드가 깨져 보임
- 앱 업데이트 확인
- 캐시 삭제 후 재시작

### 새로고침이 작동하지 않음
- 인터넷 연결 확인
- URL 소설인지 확인 (파일/텍스트는 불가)
- 파싱 룰이 유효한지 확인
