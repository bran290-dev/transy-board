## 컨텍스트란?

번역 시 AI에게 제공되는 추가 정보입니다.

| 요소 | 역할 |
|------|------|
| **캐릭터** | 이름 일관성, 인물 관계 반영 |
| **용어** | 고유명사 정확한 번역 |
| **사건** | 스토리 맥락 이해 |
| **대화** | 중요 대화 참조 |
| **요약** | 이전 회차 내용 요약 |

## 왜 중요한가?

### 컨텍스트 없이

```
田中가 말했다.
"다나카, 오랜만이야."
타나카가 대답했다.
```

→ 같은 캐릭터가 다른 이름으로 번역됨

### 컨텍스트 있을 때

```
다나카가 말했다.
"다나카, 오랜만이야."
다나카가 대답했다.
```

→ 일관된 이름으로 번역됨
