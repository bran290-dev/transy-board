고유명사와 전문용어를 관리하여 일관된 번역을 유지합니다.

---

## 용어 관리란?

작품 내 고유명사, 전문용어, 특수 표현을 등록하여 일관되게 번역합니다.

## 용어 종류

| 종류 | 예시 |
|------|------|
| **지명** | 도시, 국가, 장소명 |

## 번역 캐시 무효화

용어 변경 시 기존 번역을 업데이트합니다.

[사진: 캐시 무효화 옵션]

### 옵션

| 옵션 | 설명 |
|------|------|
| **번역 유지** | 변경 없음 |

```

모든 "성검"이 "성검 엑스칼리버"로 치환
```

## 용어 관리 팁

### 1. 일관성 유지

같은 용어는 같은 번역 사용:
```
✓ 魔法 → 마법 (통일)
✗ 魔法 → 마법/마술/주술 (혼용)
```

### 2. 작품 특유 용어 우선

일반적인 단어보다 작품 고유 용어 등록:
```
✓ 作品名의 특수 스킬명
✓ 작품 내 고유 지명
✗ 일반적인 "학교", "도시" 등
```

### 3. 설명 추가

용어 설명을 추가하면 번역 품질 향상:
```
원문: 神器
번역: 신기
설명: 신이 만든 전설의 무기, 각 나라에 하나씩 존재
```

### 4. 중복 방지

비슷한 용어가 중복 등록되지 않도록:
```
✗ 마법사, 마법사(魔法使い), 魔法使い
✓ 魔法使い → 마법사
```

## 문제 해결

### 용어가 번역에 반영 안 됨

**확인**:
1. 용어 자동 보정이 켜져 있는지
2. 원문이 정확히 일치하는지
3. 띄어쓰기, 대소문자 확인

### 잘못된 용어로 치환됨

**해결**:
1. 용어 삭제 또는 수정
2. 더 구체적인 원문 사용

### 너무 많은 용어가 자동 등록됨

**해결**:
1. 불필요한 용어 삭제
2. 중요 용어만 유지
