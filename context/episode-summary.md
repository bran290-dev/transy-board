# 회차 요약

각 회차의 줄거리와 등장 캐릭터를 관리합니다.

---

## 회차 요약이란?

각 회차의 내용을 요약하여 이후 번역에 활용합니다.

[사진: 회차 요약 탭 화면]

---

## 요약 정보

### 각 회차별 표시 항목

| 항목 | 설명 |
|------|------|
| 회차 번호 | 해당 회차 |
| 요약 | 회차 줄거리 요약 |
| 주요 사건 | 회차의 핵심 사건들 |
| 분위기 | 회차의 전반적인 분위기 |
| 장소 | 주요 배경 장소 |
| 등장 캐릭터 | 해당 회차에 등장한 인물 |

[사진: 회차 요약 항목]

---

## 자동 생성

번역 완료 시 AI가 자동으로 요약을 생성합니다.

### 생성 과정

1. 회차 번역 완료
2. 번역된 내용 분석
3. 주요 내용 요약
4. 등장 캐릭터 식별
5. 저장

### 요약 예시

```
회차: 15

요약:
주인공 철수가 마법사 길드에 가입했다.
길드 마스터와 첫 만남을 가지고,
첫 번째 임무를 받게 된다.

등장 캐릭터:
- 김철수 (주인공)
- 길드 마스터
- 접수원
```

## 요약의 역할

### 번역 품질 향상

다음 회차 번역 시:
- 이전 회차 요약 참조
- 스토리 흐름 유지
- 캐릭터 상태 반영

### 컨텍스트 윈도우

설정된 컨텍스트 윈도우만큼 이전 요약을 참조합니다.

```
현재: 20회차
윈도우: 3

참조 요약: 17, 18, 19회차
```

## 컨텍스트 삭제

회차별 컨텍스트를 삭제할 수 있습니다.

[사진: 컨텍스트 삭제 버튼]

### 삭제 대상

- 해당 회차 요약
- 해당 회차 등장 캐릭터
- 해당 회차 사건/대화

### 삭제 방법

1. 삭제할 회차 선택
2. "컨텍스트 삭제" 버튼
3. 확인 후 삭제

### 사용 사례

- 잘못된 요약 삭제 후 재번역
- 테스트 회차 정리
- 저장 공간 확보

---

## 효과적인 활용

### 1. 적절한 윈도우 크기

| 윈도우 | 효과 |
|--------|------|
| 1-2 | 빠른 번역, 적은 맥락 |
| 3-5 | 균형잡힌 설정 (권장) |
| 6+ | 풍부한 맥락, 느린 번역 |

### 2. 정기적 확인

- 요약이 정확한지 확인
- 잘못된 정보가 있으면 해당 회차 재번역

### 3. 스토리 파악

- 오래 읽지 않았던 소설 복귀 시
- 이전 내용 확인용으로 활용

---

## 제한 사항

### 요약

- 수정 미지원 (삭제 후 재번역 필요)
- 자동 생성만 지원

### 등장 캐릭터

- 자동 감지
- 수동 추가/수정 미지원

---

## 요약 품질 향상

### 좋은 요약의 조건

- 핵심 내용 포함
- 스포일러 없이 맥락 전달
- 캐릭터 행동/상태 반영

### 품질 향상 방법

1. 캐릭터 미리 등록
2. 용어 정확히 등록
3. 번역 프롬프트 최적화
