# 캐릭터 관리

소설의 등장인물을 관리하여 번역 품질을 향상시킵니다.

---

## 캐릭터 관리란?

등장인물의 이름, 별명, 설명 등을 등록하여 번역 시 일관성을 유지합니다.

[사진: 캐릭터 탭 화면]

---

## 캐릭터 목록

### 표시 정보

| 항목 | 설명 |
|------|------|
| 원문 이름 | 원어 이름 |
| 번역 이름 | 한국어 이름 |
| 역할 | 주인공, 조연 등 |
| 첫 등장 | 처음 등장한 회차 |
| 설명 | 캐릭터 설명 |

### 정렬

- 첫 등장 순
- 이름 순
- 최근 등장 순

---

## 캐릭터 추가

### 수동 추가

[사진: 캐릭터 추가 버튼]

1. 캐릭터 탭에서 "+" 버튼 클릭
2. 정보 입력
3. 저장

### 입력 필드

| 필드 | 필수 | 설명 |
|------|------|------|
| 이름 | ✅ | 캐릭터 이름 |
| 번역 이름 | ✅ | 한국어 표기 (예: 다나카 타로) |
| 별명 | ❌ | 다른 호칭들 (쉼표로 구분) |
| 역할 | ❌ | 주인공/조연/악역 등 |
| 설명 | ❌ | 캐릭터 설명 |
| 첫 등장 회차 | ❌ | 처음 나온 회차 |

[사진: 캐릭터 추가 폼]

### 별명 예시

```
원문 이름: 田中太郎
번역 이름: 다나카 타로
별명: 타로, 田中, 다나카 선배
```

→ 모든 별명이 "다나카 타로"로 번역됨

2. 이름, 첫 등장 회차 자동 등록
3. 설명은 이후 자동으로 보완

## 캐릭터 삭제

### 삭제 방법

1. 캐릭터 항목에서 삭제 버튼
2. 확인 후 삭제

[사진: 캐릭터 삭제 확인]

### 삭제 영향

- 해당 캐릭터 정보가 번역에 반영되지 않음
- 기존 번역에는 영향 없음 (일괄 치환 선택 시 제외)

## 번역 캐시 무효화

캐릭터 정보 변경 시 관련 번역을 업데이트할 수 있습니다.

[사진: 캐시 무효화 다이얼로그]

### 옵션

| 옵션 | 설명 | 용도 |
|------|------|------|
| **번역 유지** | 기존 번역 그대로 | 변경이 사소한 경우 |
| **일괄 치환** | 이름만 치환 | 이름 수정 시 권장 |
| **번역 삭제** | 재번역 필요 | 큰 변경 시 |

### 일괄 치환

예시:
```
변경 전: 타나카 → 변경 후: 다나카

"타나카가 말했다" → "다나카가 말했다"
```

## 효과적인 캐릭터 관리

### 시작 전 등록

주요 캐릭터는 미리 등록:
- 주인공
- 주요 조연
- 반복 등장 인물

### 별명 충분히 등록

캐릭터가 여러 이름으로 불리는 경우:
```
별명: 타로, 田中, 다나카 선배, 타로 군
```

### 설명 작성 팁

번역에 도움이 되는 정보:
- 성별
- 나이대
- 직업/역할
- 다른 캐릭터와의 관계
- 말투 특징

### 정기적 검토

- 10-20회차마다 확인
- 중복 캐릭터 병합
- 잘못된 정보 수정

## 문제 해결

### 같은 캐릭터가 중복 등록됨

**해결**:
1. 하나만 남기고 삭제
2. 별명에 다른 표기 추가

### 자동 보정이 안 됨

**확인**:
1. 자동 보정 설정이 켜져 있는지
2. 원문 이름이 정확한지
3. 별명이 등록되어 있는지

### 잘못된 캐릭터로 보정됨

**해결**:
1. 캐릭터 이름/별명 수정
2. 일괄 치환으로 다시 적용
