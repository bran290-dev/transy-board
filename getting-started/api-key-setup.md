# API 키 설정

Transy에서 AI 번역을 사용하려면 Gemini API 키가 필요합니다. 이 가이드에서는 API 키 발급 방법과 앱에 등록하는 방법을 안내합니다.

---

## Gemini API란?

Google에서 제공하는 AI 서비스입니다. Transy는 Gemini API를 사용하여 번역, TTS(음성 변환) 등의 기능을 제공합니다.

### Gemini의 장점

| 장점 | 설명 |
|------|------|
| **무료 티어** | 매일 무료로 사용 가능한 할당량 제공 |
| **빠른 속도** | 응답 속도가 빠름 |
| **다양한 모델** | 용도에 맞는 여러 모델 선택 가능 |
| **한국어 지원** | 한국어 번역 품질이 우수함 |

1. [Google AI Studio](https://aistudio.google.com/)에 접속

3. 프로젝트 선택 (없으면 새로 생성)
4. 생성된 API 키 복사

## 무료 티어 정보

Gemini API는 무료 티어를 제공합니다. 무료 사용량은 모델마다 다르며, Google 정책에 따라 변경될 수 있습니다.

### 무료 티어 확인 방법

최신 무료 티어 정보는 Google AI 공식 페이지에서 확인하세요:
- [Gemini API 가격 정책](https://ai.google.dev/pricing)

### 무료 티어 예시

| 모델 | 무료 할당량 |
|------|------------|
| Gemini 2.5 Flash | 일일 500 요청 (RPD) |
| Gemini 2.0 Flash | 일일 1,500 요청 |

> **팁**: 무료 티어만으로도 일반적인 소설 읽기에는 충분합니다.

1. 앱 실행 후 우측 상단 설정 아이콘 클릭
2. "API 키 관리" 섹션으로 이동

2. API 키 붙여넣기

## 어떤 모델을 선택해야 할까요?

### 처음 사용하시는 분

**Gemini 2.5 Flash lite** 를 사용하세요.
- 빠르고 품질도 좋음
- 무료 티어 사용 가능
- 대부분의 소설 번역에 적합

### 번역 품질이 중요한 분

**Gemini 3.0 Flash** 를 선택하세요.
- 더 자연스러운 번역
- 문맥 이해력이 뛰어남
- 비용이 더 높음
- 속도는 조금 느림

### 빠른 속도가 필요한 분

**Gemini 2.5 Flash Lite** 를 선택하세요.
- 가장 빠른 응답
- 비용이 가장 저렴
- 품질은 약간 낮을 수 있음

## 문제 해결

### "API 키가 유효하지 않습니다"

- 키를 복사할 때 앞뒤 공백이 포함되지 않았는지 확인
- 키가 만료되거나 삭제되지 않았는지 확인
- [Google AI Studio](https://aistudio.google.com/)에서 키 상태 확인

### "할당량 초과"

- 무료 티어 한도를 초과한 경우
- 일일 한도가 리셋될 때까지 대기 (24시간 후)
- 또는 유료 플랜으로 업그레이드
- 또는 다른 할당량을 가진 키-모델 을 추가 

### API 키를 잃어버렸어요

- Google AI Studio에서 새 키를 생성하세요
- 기존 키는 삭제하고 새 키를 사용하는 것이 안전합니다
