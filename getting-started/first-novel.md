# 첫 소설 추가하기

API 키를 등록했다면, 이제 첫 번째 소설을 추가하고 읽어볼 차례입니다.

---

## 라이브러리 화면

앱을 실행하면 라이브러리 화면이 표시됩니다. 처음에는 비어있는 상태입니다.

[사진: 빈 라이브러리 화면]

---

## 소설 추가하기

우측 하단의 **+** 버튼을 클릭하면 소설 추가 화면으로 이동합니다.

[사진: + 버튼 위치]

### 추가 방식 선택

Transy는 세 가지 방식으로 소설을 추가할 수 있습니다:

| 방식 | 설명 | 추천 상황 |
|------|------|----------|
| **URL** | 웹소설 사이트 URL 입력 | 연재 중인 웹소설 |
| **파일** | TXT/EPUB 파일 업로드 | 이미 가지고 있는 파일 |
| **텍스트** | 직접 텍스트 입력 | 짧은 텍스트, 테스트용 |

[사진: 추가 방식 선택 탭]

---

## URL로 추가하기 (추천)

가장 많이 사용하는 방식입니다.

### 1단계: URL 입력

1. **URL** 탭 선택
2. 웹소설 목록 페이지 URL 붙여넣기
3. "불러오기" 버튼 클릭

[사진: URL 입력 화면]

### 2단계: 소설 정보 확인

파싱이 완료되면 소설 정보가 표시됩니다:
- 제목
- 작가
- 설명
- 회차 목록

[사진: 소설 정보 미리보기]

### 3단계: 정보 수정 (선택)

필요한 경우 제목이나 설명을 수정할 수 있습니다.

**번역 버튼**: 제목과 설명을 AI로 번역할 수 있습니다.

[사진: 번역 버튼]

### 4단계: 저장

"저장" 버튼을 클릭하면 라이브러리에 소설이 추가됩니다.

---

## 파일로 추가하기

이미 가지고 있는 TXT나 EPUB 파일을 추가합니다.

### 1단계: 파일 선택

1. **파일** 탭 선택
2. "파일 선택" 버튼 클릭
3. TXT 또는 EPUB 파일 선택

[사진: 파일 선택 화면]

### 2단계: 회차 분리 설정

- **자동 분리**: 패턴을 감지하여 자동으로 회차 분리
- **수동 설정**: 구분 패턴 직접 지정

[사진: 회차 분리 옵션]

### 3단계: 미리보기 및 저장

분리된 회차를 미리보기로 확인 후 저장합니다.

---

## 텍스트로 추가하기

짧은 텍스트를 직접 입력하여 테스트할 수 있습니다.

1. **텍스트** 탭 선택
2. 제목 입력
3. 본문 텍스트 입력 또는 붙여넣기
4. 저장

[사진: 텍스트 입력 화면]

---

## 소설 상세 화면

소설이 추가되면 라이브러리에서 해당 소설을 클릭하여 상세 화면으로 이동합니다.

[사진: 소설 상세 화면]

### 표시 정보

- **메타데이터**: 제목, 작가, 설명, 출처
- **통계**: 총 회차 수, 다운로드된 회차, 읽기 진행도
- **회차 목록**: 각 회차의 번역 상태 표시

### 회차 상태 아이콘

| 아이콘 | 의미 |
|--------|------|
| ⬜ | 미다운로드 |
| ⬇️ | 다운로드됨 (미번역) |
| ✅ | 번역 완료 |
| 🔊 | 오디오 캐시 있음 |

---

## 소설 읽기 시작

### 이어 읽기

상단의 **이어 읽기** 버튼을 클릭하면 마지막으로 읽던 회차로 이동합니다.

[사진: 이어 읽기 버튼]

### 특정 회차 선택

회차 목록에서 원하는 회차를 직접 클릭할 수도 있습니다.

---

## 뷰어 화면

회차를 선택하면 뷰어 화면이 열립니다.

[사진: 뷰어 화면 전체]

### 자동 번역

"자동 번역" 설정이 켜져 있으면 회차를 열 때 자동으로 번역이 시작됩니다.

[사진: 번역 진행 중 표시]

### 번역 완료

번역이 완료되면 번역된 텍스트가 표시됩니다.

[사진: 번역 완료된 화면]
