# 시작하기

Transy를 처음 사용하시나요? 이 섹션에서 앱 설치부터 첫 소설 읽기까지 안내해 드립니다.

---

## 목차

1. [앱 설치](installation.md) - Play Store에서 앱 설치하기
2. [API 키 설정](api-key-setup.md) - AI 번역을 위한 API 키 발급 및 등록
3. [첫 소설 추가하기](first-novel.md) - 첫 번째 소설 추가 및 읽기

---

## 빠른 시작 가이드

### 1단계: 앱 설치
Google Play Store에서 "Transy"를 검색하여 설치합니다.

### 2단계: API 키 등록
앱 실행 후 설정에서 AI API 키를 등록합니다. (Gemini 추천)

### 3단계: 소설 추가
라이브러리에서 + 버튼을 눌러 소설을 추가합니다.

### 4단계: 읽기 시작
소설을 선택하고 회차를 클릭하면 자동으로 번역되어 표시됩니다.
