# 앱 설치

Transy를 Android 기기에 설치하는 방법을 안내합니다.

---

## 요구 사항

| 항목 | 요구 사항 |
|------|----------|
| OS | Android 8.0 (Oreo) 이상 |
| 저장 공간 | 최소 100MB 이상 |
| 네트워크 | AI 번역 및 TTS 사용 시 인터넷 연결 필요 |

---

## 설치 방법

### Google Play Store에서 설치

1. Play Store를 엽니다
2. 검색창에 "Transy" 입력
3. 앱을 찾아 "설치" 버튼 클릭
4. 설치 완료 후 "열기" 클릭

[사진: Play Store 검색 화면]

[사진: 앱 설치 화면]

---

## 최초 실행

앱을 처음 실행하면 온보딩 화면이 표시됩니다.

[사진: 온보딩 화면]

### 온보딩 단계

1. **앱 소개** - Transy의 주요 기능 소개
2. **면책조항** - 서비스 이용 약관 동의
3. **라이브러리** - 메인 화면으로 이동

---

## 권한 안내

Transy가 요청하는 권한과 그 용도입니다:

| 권한 | 용도 |
|------|------|
| 저장소 접근 | 소설 파일(TXT/EPUB) 불러오기 및 내보내기 |
| 인터넷 | AI 번역 API 호출 및 웹소설 다운로드 |

---

## 설치 문제 해결

### "기기와 호환되지 않음" 메시지가 표시됨
- Android 버전이 8.0 이상인지 확인하세요
- 설정 > 휴대전화 정보에서 Android 버전 확인 가능

### 설치가 완료되지 않음
- 저장 공간이 충분한지 확인하세요
- Play Store 캐시를 지우고 다시 시도하세요
